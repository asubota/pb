## PB | AddressBook

### Installation

Install the required tools: `yo`, `gulp`, `bower`
```
npm install -g yo bower gulp
```


Install `npm packages`:
```
npm install
```

Install `bower packages`:
```
bower install
```

## Usage

Development:
```
gulp serve
```

Build:
```
gulp
```

Unit tests:
```
gulp test
```

E2E tests:
```
gulp protractor
```
